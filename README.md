# Student Management System

A responsive Student Management System built using React, Tailwind CSS, HTML, CSS, and JavaScript. The application allows users to manage student records through a clean and user-friendly interface.

## Features

- Add student records
- View student details
- Edit existing records
- Delete student records
- Search and filter students
- Responsive design

## Technologies Used

- React.js
- Tailwind CSS
- HTML5
- CSS3
- JavaScript (ES6+)

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/student-management-system.git
````

2. Navigate to the project directory

```bash
cd student-management-system
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

## Project Structure

```text
src/
├── components/
├── assets/
├── App.jsx
├── main.jsx
└── index.css
```

This project was developed to practice React and Tailwind CSS while building a student record management application.
